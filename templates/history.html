About the History Tab:
The "File Download" tab is in place to view previous uploads to the server. This allows more accurate analysis of data and in turn better results. After files are uplaoded via the "Upload Files" tab, the file name appears here which can then be downloaded if the original somehow gets misplaced or deleted. This tab can also be used to download the generated prediction. After the file is uploaded, the generate prediction button can be pressed in the "Graph" tab. The prediction will then be placed in a file along with the originally uploaded data.
